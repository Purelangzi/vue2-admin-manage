# vue2-admin-manage

vue2+vue-router3+vuex3+element-ui+ES6
flex布局,less，echarts 
使用normalize重置浏览器默认样式
axios二次封装
使用mock模拟数据
使用js-cookie存储数据和权限控制

## 功能
- [√] 登录和登出
- [√] 主页折线图
- [√] 用户管理页列表的增删改查
- [√] 登陆权限控制
- [√] 菜单权限控制
- [√] 页面权限控制
